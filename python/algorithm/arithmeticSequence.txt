问题： 判断输入的数是否可以构成等差数列

描述： 输入一个整数N，然后输入N个整数。判断这N个整数是否可以构建一个等差数列。（0<n<1000）

输入： 第一行为一个整数N，第二行为N个整数。

输出： 可以构成等差数列输出True，否则输出False。


时间复杂度O（n）
min1=min(A)
max1=max(A)
d1=(max1-min1)/(N-1)  #等差数列的公差
if d1 == 0:
     return True
B=[0]*N
for i in A:
     n=(i-min1)/d1   #公差的n倍
     B[n]=1
return sum(B)==N  #若公差的倍数逐一递增，则说明是等差数列
